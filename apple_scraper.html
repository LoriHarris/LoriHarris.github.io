Apple Scraper

WebScraping, Python, MailChimp API, HTML

This client needed a way of notifying an overwhelming number of prospects of new listings as soon as they hit the market. The final product scrapes the clients website, determines if a new property is listed, and if so, notifies the prospect list.

The subscriber list was incorporated into the client's website and is self-maintained by MailChimp. The scraper accesses the list via the MailChimp API. The prospect receives an email with the new listing, a link to view it, and a link to unsubscribe if they are no longer in the market for housing.

This code runs via an executable on the client desktop. It visual displays if the program is running and if there are new listings or not, and states the address of any new listings.
